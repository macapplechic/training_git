Open source training

Present, share, and contribute your way to better version control

Materials

We've created an outline, a cheat sheet, and a slide deck for each class we offer. These are the same materials the GitHub Training Team uses in the classes we teach.

Teach like a GitHubber

To teach a class, just review the Outline for the class of interest, open up a web browser to the slide deck, and share the Cheat Sheet with participants for an easy-to-follow learning experience.

Contributing to these materials

Notice a typo or something that needs to be clarified in the slides, outlines, or cheat sheets? Just create a fork, make your changes, and submit a Pull Request.

Head on over to the github/training-materials repository
